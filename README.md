![Crates.io Version](https://img.shields.io/crates/v/Mp2json)

This is a small command-line program that takes as input any number of msgpack messages, concatenated, and outputs newline-delimited JSON.

It also ships with a companion program called `json2mp` which reads any number of JSON messages concatenated by newlines and outputs concatenated msgpack.
